# 게임듀오 게임 데이터 분석가 JD 분석

## 1. 회사/포지션 개요
- 게임듀오: 모바일 게임 개발/퍼블리싱사 (아처키우기, 캣히어로, 어나던던전 등)
- 매출 추이: 2020년 40억 → 2021년 100억 → 2022년 270억 → 2023년 606억 → 2024년 631억 (급성장)
- 채용 전형: 서류 → 인터뷰 → 처우협의 → 최종합격 (서류에서 자격요건/지원동기 확인)

## 2. 주요 업무 (5가지 축)

| 영역 | 내용 |
|---|---|
| 데이터 정제/가공/분석/자동화 | 유저 행동 Raw 데이터 정제·가공, 리포트/대시보드 제작 → Action Plan 연결. 반복 리포트 자동화, 지표 시스템화 |
| 핵심 지표/로그 설계 및 실험 분석 | 지표/로그 설계·관리, A/B 테스트 등 가설 검증 실험 기획·진행 |
| 데이터 기반 의사결정 | 경영진/PD/기획/개발/마케팅과 협업, Data-Informed 문제해결 제시 |
| ML/DL 활용 | 예측 모델 개발 및 운영 반영, 유저 경험 개선 |
| 데이터 문화 개선 | 분기 1-2회 데이터 세미나 기획, 인사이트 아티클 작성 |

## 3. 자격요건 (필수)

- **데이터 핸들링**: 로그 설계~최종분석 전체 프로세스 경험, Python·SQL 활용 데이터 가공/설계
- **분석 인프라**: GCP(BigQuery)/AWS 등 클라우드 대용량 데이터 경험, Looker Studio·Tableau 등 BI Tool 활용
- **문제해결**: A/B Test 설계 및 가설 검증을 통한 비즈니스 인사이트 도출, 주도적 오너십

## 4. 우대사항

- 게임 장르/BM(비즈니스 모델) 이해, 게임 핵심지표(LTV, ARPU, CPI 등) 분석 이해도
- 인과추론(Causal Inference) 방법론 실무 적용 경험
- Airflow, dbt 등 데이터 파이프라인 자동화/운영 경험
- 생성형 AI를 분석/리포팅에 접목한 생산성 향상 경험

## 5. 포트폴리오 매핑 — 어떤 프로젝트가 필요한가

JD가 요구하는 역량을 커버하려면 아래 요소를 프로젝트에 녹여야 함:

1. **모바일 게임 유저 행동 로그 데이터** (가상/공개 데이터셋) 정제·가공 파이프라인
   - Python/SQL(BigQuery 문법) 사용
2. **핵심 지표 설계 및 대시보드**
   - DAU/MAU, 리텐션(D1/D7/D30), LTV, ARPU/ARPPU, CPI, 전환율 등 게임 지표 정의·산출
   - Looker Studio 또는 Tableau로 시각화
3. **A/B 테스트 설계 및 분석**
   - 가설 수립 → 실험 설계 → 통계적 유의성 검정 → 인사이트 도출 (필요시 인과추론 기법: DID, PSM 등)
4. **이탈(Churn) 예측 or LTV 예측 ML 모델**
   - ML/DL 활용 우대사항 대응
5. **자동화 파이프라인**
   - Airflow/dbt로 리포트 자동화 (우대사항 대응, 가산점)
6. **생성형 AI 활용 사례**
   - 리포트 요약/인사이트 생성에 LLM 접목 (차별화 포인트)
7. **인사이트 아티클/리포트 문서화**
   - 데이터 세미나/아티클 작성 역량 어필용 결과 보고서 1편

## 6. 다음 단계 제안

- 프로젝트 스코프 및 일정 확정
- 기술 스택 확정 (Python/SQL/BigQuery/Looker Studio 등)
