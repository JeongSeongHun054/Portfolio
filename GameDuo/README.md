# 던전키우기 게임 데이터 분석 포트폴리오

게임듀오 "게임 데이터 분석가" 채용 공고에 대응하기 위해 만든 포트폴리오입니다. 실존 게임 데이터 대신, 게임듀오가 서비스하는 게임과 같은 방치형(Idle) RPG를 가정해 유저 50만 명·365일 규모의 데이터를 직접 설계하고 실제 GCP BigQuery 위에서 생성·분석했습니다.

## 핵심 요약

50만 유저 시뮬레이션 데이터로 세션 736만 건, 이벤트 9,202만 건, 매출 292만 달러 규모의 데이터를 실제 BigQuery 환경에서 처리했습니다. 리텐션 곡선(D1 47.3% / D7 20.7% / D30 6.3%), A/B 테스트 2건(가챠 확률 부스트 실험은 p<0.000001로 유의미, 튜토리얼 개편 실험은 p=0.64로 유의미하지 않음), BigQuery ML 기반 이탈 예측 모델, dbt/Airflow 자동화 설계까지 포함되어 있습니다.

## 기술 스택

GCP BigQuery (SQL, BigQuery ML, 파티셔닝/클러스터링) · Python · dbt · Airflow · Chart.js
